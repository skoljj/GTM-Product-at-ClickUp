Three-Layer Stack — Modular Breakdown

Each module is a standalone Three.js scene you can orbit, inspect, and iterate on independently.
Module 01
Contributor Layer
38 buildings across 6 districts (Commercial, Enterprise, Partnerships, Post Sales, XDR, Operations). Fountain with particle system. 228 ICs mapped to 6-floor buildings.
→
Module 02
ClickUp 3D Glyph
Floating 3D ClickUp chevron logo with purple/pink gradient arms, apex vertex, aura rings, and orbital rotation animation.
→
Module 03
Product Layer Disc
Circular disc shell with 3 internal zones — API Layer (ceiling), Codebase (core ring), MCP Layer (floor). Edge rings, zone labels, interior particles.
→
Module 04
API Layer
5 LLM partner pods (OpenAI, Anthropic, Gemini, Grok, Perplexity) + Manus. Stacked rotating discs, gateway rings, outward beams into deep space, ambient particle field.
→
Module 05
Codebase — Studio + Lab
Core ring split into Studio (Create) and Lab (Analyze). Immersive Coderoom with holographic code walls, blinking cursor, ceiling lights, floating data motes.
→
Module 06
MCP Gears
Floor zone with 4 interlocking gear meshes — ClickUp (native), Salesforce, Gong, Figma. Counter-rotating at varied speeds with teeth detail and inner rings.
→
Module 07
Agent Layer
Execution universe: 6 activity tiles mapped to districts. Activity cubes with particle swarms (800–1400 agents), super agent characters (Scout, Analyst, Builder, Advisor, Orchestrator).
